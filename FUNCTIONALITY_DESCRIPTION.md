# Описание функционала приложения Coach Fit

## Общий функционал
- Регистрация и авторизация (клиенты и тренеры)
- Подтверждение телефона через SMS
- Онбординг клиентов (сбор данных, цели, ограничения)

## Функционал для клиентов
- Дашборд со статистикой тренировок и метриками
- Календарь тренировок с возможностью создания, редактирования, повторяющихся событий
- Программы тренировок (просмотр от тренера, создание своих)
- Отслеживание метрик тела и упражнений с графиками
- Дневник питания
- Профиль и настройки

## Функционал для тренеров
- Управление клиентами (список, добавление, редактирование данных, статистика)
- Календарь команды (просмотр и создание тренировок для всех клиентов)
- Программы тренировок и библиотека упражнений
- Финансы (учет платежей, статистика доходов, аналитика)
- Просмотр метрик и программ клиентов

## Технический стек
- Backend: FastAPI (Python), PostgreSQL, JWT-аутентификация
- Frontend: React + TypeScript + Vite
- Docker для развертывания
